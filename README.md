# tcsamSurveyData

An R package with functions related to processing AFSC bottom trawl survey data for the Tanner crab assessment.

## Required R packages

Package sources available as git repositories under <https://github.com/wStockhausen/>:

-   tcsamFunctions
-   wtsGIS
-   wtsPlots
-   wtsUtilities

Available on CRAN:

-   dplyr
-   ggplot2
-   magrittr
-   plyr
-   readr
-   reshape2
-   rlang
-   scales
-   sf
-   sqldf
-   stats
-   tmap
-   utils
